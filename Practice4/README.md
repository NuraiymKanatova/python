# Practice 4: Python Advanced Topics - Iterators, Dates, Math, and JSON

## Topics Covered

### Iterators and Generators
**Iterators**
- Iterators: `iter()` and `next()`
- Loop through an iterator
- Create an Iterator

**Generators**
- `yield` keyword
- Creating Generator Functions
- Generator Expressions

### Dates and Time
- `datetime` module
- Creating Date Objects
- Date Formatting
- Calculating Time Difference (`timedelta`, subtraction, `.total_seconds()`)
- Working with Timezones (`timezone`, UTC, offsets)

### Math
**Built-int**
- `min`, `max`, `abs`, `round`, `pow`

**Math Module**
- `sqrt`, `ceil`, `floor`, `sin`, `cos`, constants `pi`, `e`

**Random Module**
- `random()`, `randint()`, `choice()`, `shuffle()`

### JSON
- JSON syntax (strict formatting rules)
- Parsing JSON: `json.loads()`
- Converting Python to JSON: `json.dumps()`
- Writing JSON files: `json.dump()`
- Reading JSON files: `json.load()`
- Working with JSON data (`sample-data.json`)

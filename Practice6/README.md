# Practice 6 - Python File Handling, Directory Management, and Built-in Functions

## Overview
This project contains practical Python examples for:

- File handling
- Directory management
- Built-in functions

It is based on study topics such as reading/writing files, wotking with folders, and using useful Python built-in functions.

## Topics Covered

### 1. File Handling
- File models: `r`, `w`, `a`, `x`
- Reading files with:
  - `read()`
  - `readline()`
  - `readlines()`
- Writing and appending data
- Using `with` statement
- File/path operations with `pathlib` and `shutil`

### 2. Directory Management
- Creating folders with:
  - `os.mkdir()`
  - `os.makedirs()`
- Listing directory contents with `os.listdir()`
- Changing current directory with `os.chdir()`
- Checking current working directory with `os.getcwd()`
- Removing empty directories with `os.rmdir()`
- Finding files by extension
- Moving/copying files

## 3. Built-in Functions
- `len()`
- `sum()`
- `min()`
- `max()`
- `map()`
- `filter()`
- `reduce()` from `functools`
- `enumerate()`
- `zip()`
- `sorted()`
- Type checking and conversions

# Practice 3 - Advanced Python Concepts

This practice contains structured examples based on advanced Python topics.

The goal of this practice is to demonstrate understanding of:

- Functions
- Classes and Objects
- Inheritance
- Lambda Functions

Each topic is organized into a separate folder with detailed subtopic examples.

## Topics Covered

### 1. Functions
Examples include:
- Function arguments
- Positional and Keyword arguments
- Arbitary arguments (*args)
- Arbitary keyword arguments (**kwargs) 
- Return values
- List as an argument
- Unpacking arguments

### 2. Lambda Functions
Examples include:
- Basic lambda syntax
- Lambda using map() function
- Lambda using filter() function
- Lambda using sorted() function

### 3. Classes and Objects
Examples include:
- Basics of classes and objects
- __init__() method
- Self parameter
- Class variables and instance variables
- Modifying and deleting object properties

### 4. Inheritance
Examples include:
- Parent and Child classes
- Super() function
- Multiple inheritance
- Add methods
- Add properties

# Music Player with Keyboard COntrols

## Description
This project is an interactive music player built using Python and Pygame.

The application allows users to play audio tracks, control playback using the keyboard, and view the current track and playback progress.

## Features 
- Play, stop, next, and previous track controls
- Keyboard-based interaction
- Displays current track name
- Shows playback progress (time and progress bar)
- Automatically loads tracks from a folder (playlist management)
